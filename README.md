### Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/shivam-code-hub/assesment-youtube.git
    ```

2. Create a `.env` file inside the repository:

    ```bash
    touch .env
    ```

3. Copy the content from `env_example.txt` file to `.env` file.

4. Build and run the Docker containers:

    ```bash
    docker-compose build --no-cache
    docker-compose up -d
    ```

## Testing

- Use the Postman collection attached (`postman_collection.json`) to test the APIs.
